" What is turn.js? "



  • JavaScript Library Plug-in
  • Replicates the feel of turning pages on an actual book
  • No Flash needed
  • Implements Modinizr.js

" How does it work? "



  • Each page is assigned to its own div
  • By using a number naming convention it makes things easier to call
  • A loadApp() function is implemented to set widths/heights
  • CSS needs the widths/heights to be tweeked as well
  • Modinizr.js is used to have polyfill fallbacks for browsers that cannot support it

Implementing loadApp() function


function loadApp() {

    // Create the flipbook

        height:700,

        gradients: true,
        autoCenter: true

Implementing Keyboard Navigation


    // Keyboard arrow navigation
    $(document).keydown(function(e){
        var previous = 37, next = 39;

        switch (e.keyCode) {
            case previous:  $('.flipbook').turn('previous');
                            break;
            case next:      $('.flipbook').turn('next');
                            break;
        }
    }); 

} //End loadApp() function
